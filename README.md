# Portfolio & Personal Website

This is my personal website, housing my portfolio, a blog and some articles, and a collection
of interactive tools for making or teaching music and game development.

## Usage

This project uses astro, here are some commands :

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | Installs dependencies                            |
| `npm run dev`             | Starts local dev server at `localhost:4321`      |
| `npm run build`           | Build your production site to `./dist/`          |
| `npm run preview`         | Preview your build locally, before deploying     |
| `npm run astro ...`       | Run CLI commands like `astro add`, `astro check` |
| `npm run astro -- --help` | Get help using the Astro CLI                     |

Feel free to check [Astro documentation](https://docs.astro.build)
